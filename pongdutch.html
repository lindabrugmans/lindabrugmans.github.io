PONG

korte uitleg Project PONG

Dit project was een van de eerste projecten die ik gemaakt heb. Toen ik gestart was met dit project had ik geen ervaring met progammeren, UNITY, of enkele andere codeer talen.

Wat was het doel van dit project?

Het doel van dit project was om de basics van UNITY en C# te leren kennen, en om te kijken of ik er zelfstandig uit zou kunnen komen.

Wat heb ik geleerd met dit project?

1) De basis van UNITY --> hoe je een project aanmaakt, een script, image, assets toevoegd.

2) Basis code van C#

3) Niet opgeven ookal werkt het niet op de manier dat je het wou laten werken.

Conclusie

Bij dit project voelde het alsof ik in het diepe gegooit werd. Wat nu ik er zo op terug kijk ook zo was, aangezien ik geen ervaring had. Dit is ook een reden waarom het een goed gevoel gaf toen ik het eind resultaat zag. Aangezien het project werkte, en het mijn eerste project was.
