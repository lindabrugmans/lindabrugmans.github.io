Project ToDoApp

Explanation
The idea of this project was to create a ToDoApp. The main purpose of this application was for the user to be able to add or remove tasks if these were finished.
Learning Goals
- Improving knowledge on UNITY
- Working with buttons
- Get to know a few basic lines of C#
Conclusion
This project made me realize that UNITY doesnt have to be for games only. The possilibities with UNITY are bigger than I had thought. It was interesting, because I also got to work with C# again, but it also made me realize that I personally think working with websites is more fun.
