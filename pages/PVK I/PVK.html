This project was my first time working with Backend. I've worked with Mongodb, and Express. The idea was to create a 'Portfolio Maker' in which you can eventually add your projects easily by just pressing a button on the website. It will automatically add the new project; containing a project name, date, and description.

I've worked with Mongodb, Express, VUE and Postman. I created an API which I fetched in my VUE project. Where I eventually created 2 of the CRUD functions: Create and Read. This project is not finished yet, since it only has a basic layout and only 2 of the 4 needed functions.
