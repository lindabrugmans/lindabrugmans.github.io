Informatie

Dit project was de eerste keer dat ik gewerkt heb met Backend. Ik heb gewerkt met MongoDB en Express. Het idee was om een 'portfolio maker' te maken. Waarin je makelijk doormiddel van een knop projecten toe kan voegen. Het zou automatisch een nieuw project toevoegen. Met daarin een project naam, datum en een omschrijving.

Ik heb gewerkt metMongodb, Express, VUE en Postman. Ik heb een API gemaakt die ik vervolgens aangevraagd heb in mijn Vue project. Waarin ik uiteindelijk 2 CRUD functies gemaakt heb: Create en Read. Dit project is nog niet afgerond. Het heeft een simpele layout en 2 van de 4 functies.
