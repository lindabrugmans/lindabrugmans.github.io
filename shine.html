Project Shine

Explanation

Learning Goals
- Making a "portfolio"
- Work with Visual Studio Code
- Increase knowledge on HTML

- Increase knowledge on CSS

Conclusion
I really liked this project, because it was all pretty new to me. And because of that I learned quite some things, which made me realize that there are plenty more options with websites. This project made me realize that in the future I want to work with websites.
