TODOAPP

Small explanation Project TODOAPP

The idea of this project was to create a ToDoApp, also known as some sort of planner. And the main purpose of this Application was that the user is able to add or remove tasks incase these were finished.

What was the goal of this project?

Create an application which you can personally use, or just some extra experience with working in UNITY.

What have I learned in this project?

1) UNITY doesnt have to be for games only

2) Basic C#

3) Basic UNITY

Conclusion

This project made me realize that UNITY doesnt have to be for games only. The possilibities with UNITY are bigger than I had thought. It was interesting, because I also got to work with C# again, but it also made me realize that I personally think working with websites is more fun.
