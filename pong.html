Small explanation Project PONG

This was one of the first projects that I have made. When starting this project I had zero experience with coding, UNITY or any language at all.

What was the goal of this project?

The purpose of this project was to get to know UNITY and the basics of C#.

What have I learned in this project?

1) The basics of UNITY --> How to make a project, create a code, add images and assets.

2) A few basic lines of code of the language C#.

3) To not give up even when something does not work the way I wanted it to work.

Conclusion

In this project it felt like I had been thrown into the deep. Which was the case, because I had zero experience. This is another reason why it felt good to see the end results. Because it was a working Pong game, and my first ever project.
