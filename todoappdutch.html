TODOAPP

Korte uitleg Project TODOAPP

Het idee van dit project was om een ToDoApp te maken. Eigenlijk dus een soort planner. de bedoeling was dat de gebruiker uiteindelijk

Wat was het doel van dit project?

het doel van dit project was om een programma te maken waarin je kan plannen. de "ToDoApp" had een aantal eisen, je moest zoal taken toe kunnen voegen, en kunnen verwijderen als deze afgerond waren

Wat heb ik geleerd in dit project?

1) UNITY hoeft niet alleen gebruikt te worden voor Games

2) Basis C#

3) Basis UNITY

Conclusie

In dit project realizeerde ik mij dat UNITy niet alleen gebruikt hoeft te worden voor games. Het was interresant om te zien dat de mogelijkheden met UNITY groter zijn dan ik dacht. Maar ik realizeerde me ook dat ik websites toch leuker en interresanter vind om mee te werken.
