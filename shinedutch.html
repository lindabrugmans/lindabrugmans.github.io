Korte uitleg Project SHINE

Bij dit project was de bedoeling om een website te maken. Dit was de tweede website die ik ooit gemaakt heb. Bij de eerste website was het gewoon een website om te basis te leren van hoe je een website maakt.

Wat was het doel van dit project?

Het doel in dit project was om een website te maken en je projecten daar in te zetten die je ooit gemaakt hebt.\ Dit project werd toen gebruikt als een portfolio.

De website had een paar eisen, zoals --> je naam, contact form, en projecten.

What have I learned in this project?

1) Wat ik later wil gaan doen --> Websites

2) Basis HTML

3) Basis CSS

Conclusie

Ik vond dit project leuk om te doen, ook omdat het allemaal nieuw was voor mij. Doordat het nieuw was heb ik hierdoor ook veel geleerd, en dat heeft mij laten realizeren dat je ook met websites veel meer kanten en opties hebt. Met dit project ben ik ook tot de keuze gekomen dat ik in de toekomst met websites wil gaan werken.
