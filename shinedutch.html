Project Shine

Korte uitleg
Het doel van dit project was het maken van een website. Dit was de tweede website die ik ooit heb gemaakt. De eerste website ooit was om te leren wat een website is en hoe het gemaakt kan worden.
Leer doelen
- Een "portfolio" maken
- Werken met Visual Studio Code
- Vergroot kennis over HTML
- In staat zijn om een website met meerdere pagina's te maken
- Vergroten van kennis over CSS
- In staat zijn om je website er beter uit te laten zien; niet gewoon wit.
Conclusie
Ik vond dit echt een leuk project, omdat het allemaal vrij nieuw voor me was. En daardoor heb ik heel wat dingen geleerd, waardoor ik me realiseerde dat er nog veel meer mogelijkheden zijn met websites. Dit project heeft me doen beseffen dat ik in de toekomst wil gaan werken met websites.
